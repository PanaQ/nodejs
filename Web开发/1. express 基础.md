express是什么？
===
Fast, unopinionated, minimalist web framework for node.

1. 快速
2. 无偏见
3. 最小化的
4. Web框架

---
最简单的示例
===
```
var express = require('express')
var app = express()

app.get('/', function (req, res) {
  res.send('Hello World')
})

app.listen(3000)
```

安装
===
```
npm install express
```
---
使用生成器
===
1. 安装
```
npm install -g express-generator@4
```
2. 创建应用
```
express /tmp/foo && cd /tmp/foo
```
3. 安装依赖
```
npm install
```

---

4. 启动服务器

```
npm start
```
